# KeFu
在线客服系统，开发语言：NodeJS。通信库：Socket.IO。UI库：前端SUI，后端LayUI。

# 缘起
公司商城里面想加个在线客服，但是第三方的的太贵，功能太多又又不上。所以闲暇时间写了这个，让公司能暂时用起来。
估计还是有bug，未详细测试。

# 使用方法
git clone https://github.com/gytai/KeFu.git

cd KeFu

npm install

pm2 start bin/www --name kefu
